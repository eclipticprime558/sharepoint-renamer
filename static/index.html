What you'll need

M365 Account

A Microsoft 365 account with access to the SharePoint site you want to rename files in. Admin access required for the first-time setup.

SharePoint Site URL

The URL of your SharePoint site, e.g.
https://yourorg.sharepoint.com/sites/SiteName

Library Name

The name of the document library to scan, e.g. Documents or SharePoint Server. You can find this in SharePoint under Site Contents.

Azure App Registration

  1. Open Azure Portal → App Registrations → New Registration
  2. Name: anything you like (e.g. SharePoint Renamer)
  3. Supported account types: Accounts in any organizational directory (Multitenant)
  4. Redirect URI: choose Single-page application (SPA) and paste this URL:
  5. Click Register, then copy the Application (client) ID from the Overview page
  6. Go to API permissions → Add a permission → Microsoft Graph → Delegated and add:
    Sites.ReadWrite.All  and  Files.ReadWrite.All
  7. Click Grant admin consent (requires admin role)
From the Overview page of your Azure App Registration. Stored locally in your browser.
Connect to Microsoft 365

Sign in with your Microsoft 365 account. The first time you connect, Microsoft will ask you to consent to the permissions your app registration requests.
